## 이번 방학 (~2026-09-07 개강)

**기준: "긴 시간이 통째로 필요한 것"만 여기서 한다.** 짬짬이 되는 건 학기로 미룬다.

- [x] **블로그 마무리** — 30분 ✅ 2026-07-22
  - 개발일지 #17 발행 완료 → 연재 17편 (라이브 `total:17, index:17` 확인)
  - backend deps PR 병합 + 배포 완료 (그 과정에서 bcrypt 5.0 회귀를 배포 전에 잡음)
- [x] **복원 훈련 (DR drill)** — 반나절 ✅ 2026-07-22 (`scripts/restore_drill.sh`)
  - 개수 대조를 넘어 **구조 검사**로 판정한다: 테이블 집합·인덱스·제약·시퀀스·
    alembic 버전·확장·쓰기 가능·BYOK 복호화. 행 수는 백업 이후 쓰기가 있으면
    정상인데도 어긋나므로 방향으로 분류(DRIFT)하고 불합격시키지 않는다.
  - **측정값**: 복원 656ms, 전송 포함 RTO 약 2초. RPO는 '마지막 정지 시점'
    (백업이 cron이 아니라 정지 절차에서 돈다).
  - **면접에서 쓸 것**: 여기서 나온 게 "훈련을 만들었다"가 아니라 **훈련이
    통과시키던 13가지 구멍**이다 — 테이블 12개 중 5개만 보고 있었고, 키가 없어도
    "통과"로 찍는 오판정이 있었고, 실패하면 훈련용 DB가 운영 서버에 남았다.
  - 더 나아간 것: `RECOVERY.md` 런북 + **임시 인스턴스로 맨바닥 재건 리허설**.
    거기서 문서 버그 2개를 찾았다(buildx 미설치로 빌드 불가 / `.dockerignore`
    누락으로 시크릿이 이미지에 구워지고 앱이 재시작 루프).
- [x] **알림** — 반나절 ✅ 2026-07-22 (`scripts/watch.sh` + `.github/workflows/watch.yml`)
  - CloudWatch+SNS가 아니라 **매시 도는 GitHub Actions**로 만들었다. 이유가 있다:
    이 서버는 필요할 때만 켜므로 "백엔드가 죽었다"를 그냥 알람으로 걸면 **끌 때마다
    울린다**. 진짜 이상은 "EC2가 켜져 있는데 공개 API가 죽음"이라는 **조합**이고,
    그건 AWS 안이 아니라 **바깥에서** 봐야 안다 — 앱 자가점검은 `backend_ok`가
    하드코딩 `True`라 27,506번 중 실패가 0건이었고 CloudFront를 아예 못 본다.
  - 지금 보는 것: EC2 running인데 공개 API가 죽었는가 / 6시간 이상 켜둔 채 방치 /
    최신 백업 나이 / 만료 안 되는 사본 / 이미지 사본 / SES 샌드박스 여부.
  - **효과 실증**: 첫 실행에서 실제 문제 2건을 그대로 잡았다 — 오리진이 주차된 채
    3시간 방치된 것과, **SES 프로덕션 액세스가 4주째 거부 상태**인 것.
  - 남은 것: AWS 네이티브 경로(SNS + EC2 상태검사 알람)를 추가하면 인스턴스 자체가
    죽는 경우를 1시간이 아니라 몇 분 안에 안다. 무료 티어 안에서 가능.
- [ ] **컨테이너 오케스트레이션 (ECS)** — 2~3주 ← 방학의 본체
  - 처음이면 1~2주로 안 끝난다고 보는 게 안전. 특히 헬스체크가 안 붙어 태스크가 죽는 구간
  - **비용 주의**: ALB가 월 ~$16 고정. "안 쓸 때 끈다"가 안 통한다.
    → 크레딧(2027-06-24 만료) 안에서 만들고, 근거(스크린샷·일지) 남기고 정리하는 방식 권장
  - 대안: 로컬 k3s/kind면 $0이지만 "실제 서비스를 옮겼다"는 서사가 약해짐

**+ 매일 자투리: SAA 인강·문제풀이 1시간**
ECS와 시험범위가 겹친다(VPC·서브넷·ALB/타깃그룹·헬스체크·Auto Scaling·IAM 역할).
안 겹치는 부분이 공부할 곳: RDS Multi-AZ/복제본, DynamoDB, SQS/SNS/Kinesis,
Route 53 라우팅 정책, EFS, S3 스토리지 클래스, Organizations.
→ 목표: 8월 말~9월 초 응시. **날짜를 먼저 예약**하면 밀리지 않는다.

   (백업 미실행 / 주석이 없는 방어를 있다고 서술 / 모델 교체 시 조용한 기본값 변화).
